\chapter{气功}

\section{《阴符经》}

\paragraph{宋 张伯端}
\begin{quote}
《阴符》宝字逾三百，《道德》灵文止五千；\\
今古上仙无限数，尽从此处达真诠。
\end{quote}

\begin{quote}
  火生于木，祸发必克。
\end{quote}

\begin{itemize}
  \item 上篇：学道修炼，必须尽心观执，奉行天道，天人相应，万化虽多，不出一身之内，精华身心，方可到达养生修炼的最高境界。
  \item 中篇：天地吸收万物之气而成其体，万物赖人气之生养而成其性，人摄万物之气以养其身。
  \item 下篇：学道修炼者，必须戒目收心，无邪妄动，自然之道不可违；养之太过，役心损虑，反而招至损命。
\end{itemize}
