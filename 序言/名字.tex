\chapter{名字}

\paragraph{观}
\begin{itemize}
  \item 我只看看，不说话
  \item 生慧\footnote{止生定，观生慧，参见止观法门篇章}
\end{itemize}

\paragraph{澜}
书海；取明代藕益智旭大师的《法海观澜》之意。

\paragraph{札}
笔记；书札。

\paragraph{半夏}
\begin{itemize}
  \item 一味中药名\smiley。
  \item 夏虫不可以语冰\footnote{语出《庄子集释》卷六下《外篇·秋水》。比喻人囿于见闻，孤陋寡闻，知识短浅，不通时务。}
\end{itemize}
